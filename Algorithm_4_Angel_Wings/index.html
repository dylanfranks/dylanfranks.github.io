Description

Algorithm for Angel Wings is a music generation game for Windows and HTML5. It expands upon the procedural music experiments Dylan began in 2016, while also streamlining it in a more user-friendly format.

This game supports touchscreen input, and can be played on mobile as well as in your browser.

You play the game by catching grey cells with the cursor before they disappear. You can catch cells in quick succession to rack up combos. Music is randomly generated as you play.

Features

  • Procedurally generated music
  • 13 different instruments (piano, harp, synth, choir, etc)
  • 12 scales (major, minor, whole tone, chromatic, etc)
  • 12 musical keys
  • 10 time signatures that each create wildly different patterns
  • Autoplay feature generates music endlessly
